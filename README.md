## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/Winning-Corporation-React.git
```

Go to the project directory

```bash
  cd winning-corporation-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
